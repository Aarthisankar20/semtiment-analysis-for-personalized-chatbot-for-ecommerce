his project implements a personalized chatbot for e-commerce platforms using Sentiment Analysis and Natural Language Processing (NLP). The system analyzes customer reviews, determines product sentiment, and recommends highly rated products to users. By summarizing customer opinions and highlighting top-rated items, the chatbot reduces the effort required to read multiple reviews and enables faster purchasing decisions.
